# 历年电赛 K230 赛题整理

本目录包含了 2021, 2023, 2024 年的整理资料，以及 2025 年的原创代码。

## 目录结构

*   **00_Reference/**: 参考资料，包含 STM32 相关代码等。
*   **01_2021_F_DrugCar/**: 2021年 F题 - 智能送药小车。
*   **02_2023_E_Laser/**: 2023年 E题 - 运动目标控制与自动追踪系统。
*   **03_2024_E_ThreePieceChess/**: 2024年 E题 - 三子棋游戏装置。
*   **04_2025_E_SelfAiming/**: 2025年 E题 - 简易自行瞄准装置 (原创)。

## 2025年 E题 (简易自行瞄准装置) 结构说明

2025年的代码位于 `04_2025_E_SelfAiming` 目录下，主要结构如下：

### 1. Result (最终运行结果)
该目录存放经过调试、可以直接运行的主程序脚本。针对不同的比赛场景需求，提供了多个版本的 `main.py`：
*   **main（可阈值调节高帧）.py**: 优化了帧率，适合需要快速响应的场景。
*   **main（可阈值调节高识别）.py**: 优化了识别准确率，适合对精度要求较高的场景。
*   **main（多限制条件）.py**: 增加了更多的判断逻辑和限制条件，提高抗干扰能力。

### 2. find_rect (矩形识别模块)
该目录包含与矩形识别相关的训练模型和部署代码：
*   **mp_deployment_source/**: 存放训练好的 kmodel 模型文件 (`.kmodel`) 和部署配置文件 (`deploy_config.json`)。
*   **det_results/**: 存放识别测试的结果图片。
*   **det_image_*.py / det_video_*.py**: 用于测试模型在图片和视频上的识别效果。

### 3. 方法及参考 (迭代过程与参考)
该目录记录了开发过程中的中间版本和参考资料：
*   **迭代版本/**: 存放开发过程中尝试过的各种算法和逻辑，如“自定义矩形判断”、“投票选矩形”、“激光点预测”等。这些代码展示了从初步尝试到最终方案的演进过程。
*   **NUEDC-2025-E-master/**: 包含激光矩形调参等工具，辅助调试。
*   **ERROR/**: 存放开发过程中遇到错误或废弃的代码片段。

### 4. 题目文档
*   **E题_简易自行瞄准装置.pdf**: 2025年电赛 E题的官方题目说明文档。
